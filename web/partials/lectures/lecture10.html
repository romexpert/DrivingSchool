ПРОЕЗД ПЕРЕКРЕСТКОВ
 
n Очередность проезда определяется светофором, знаками приоритета или отсутствием помех
n Поворот направо - из крайней правой в крайнюю правую полосу. Поворот налево - из крайнего левого положения в любую из полос своего направления - VIII (расположение на крайней левой полосе разрешает разворот на перекрестке). Такой порядок определяется исходя из нашей посадки с левой стороны в автомобиле.
n  Если покрытие дороги не определяется (темень, снег и т.д.) или не видно знаков приоритета, то дорога априори считается второстепенной (неверно определенный приоритет может привести к аварии).
n При заторах на перекрестке разрешен выезд на него только для движения в свободных направлениях (т.е. не перекрывая движения на перекрестке).
n Разворот на перекрестке (как и вне его) должен быть произведен за один прием (без применения заднего хода, и последовательно пропуская возникающие помехи справа).
n На перекрестке разрешены (при отсутствии ограничений) четыре маневра: 1) направо, 2) прямо, 3) налево, 4) разворот.
 
n  РП - Регулируемый перекресток:
-  при повороте налево на зеленый светофор, уступить на середине перекрестка дорогу т.с., двигающимся навстречу (помеха справа);
-   приоритет т.с., завершающего маневр на перекрестке;
-   приоритет трамвая в равных условиях;
-   перед завершением поворота налево, остановка у стоп - линии перед красным светофором в разрыве разделительной полосы (т.е. при проезде руководствоваться стоп - линиями  и светофорами на пути следования);
-  запрет обгона с выездом на встречную полосу;
-  пропустить т.с. с других направлений при движении под основной красный сигнал светофора и зеленую стрелку в дополнительной секции.
 
n  НРП - Нерегулируемый перекресток:
-   приоритет т.с., не имеющего помехи справа;
-   приоритет трамвая в равных условиях;
- въезжающий по второстепенной дороге, пропускает т.с. с главных дорог (обладают приоритетом вне зависимости от направления их дальнейшего движения);
-  въезд со второстепенной дороги по «непересекающейся» траектории только при условии, что не будет создано помехи (движение в свободную полосу, разъезд бортами);  т.е., если траектория т.с. с главного направления не совпадает с нашей - мы начинаем движение;
-  приоритет поперечного разъезда дает преимущество въезда на перекресток (действует до его середины), а приоритет встречного разъезда дает преимущество выезда с перекрестка; т.е. при движении через перекресток следим персонально за «своей» помехой справа (за всеми уследить невозможно) на участке до середины (м.б. промежуточная остановка), а затем - до выезда с перекрестка.
 
 
ПЕШЕХОДНЫЙ ПЕРЕХОД И ОСТАНОВКИ МАРШРУТНЫХ Т.С.
 
n  Водитель обязан пропустить пешеходов, переходящих проезжую часть по ПП, а при проезде по нешироким улицам (в две полосы) должен быть готовым к появлению пешеходов в не установленных для перехода местах (препятствовать переходу нельзя).
n    При включении зеленого водитель обязан пропустить пешеходов, завершающих переход.
n   Если т.с. перед ПП ограничивает его обзор, следует притормозить, чтобы убедиться в отсутствии перед т.с. пешеходов.
n Технологическая остановка (н-р, при заторах) разрешена только непосредственно перед пешеходным переходом, но не на зебре (не создавать помехи движению пешеходов).
n Преднамеренная остановка разрешена за 5 м до перехода или непосредственно за ним (не перекрывается обзорность перехода).
n  Перед т.с. со знаком «дети» - следует снизить  скорость, предполагая возможную остановку при появлении детей из-за т.с.
n  На ПП движение только прямо. Запрещены: обгон, остановка, движение задним  ходом, разворот.
n При посадке пассажиров в трамвай с проезжей части - обязательно пропустить пешеходов, идущих к остановившемуся трамваю и от него (при их отсутствии, при еще не закрытых дверях - можно продолжить движение, соблюдая необходимую предосторожность и не форсируя движения). При посадке пассажиров в трамвай с разделительной полосы обязательно только снижение скорости при проезде мимо такой остановки.
n В н.п. отъезжающий от обозначенной остановки автобус и троллейбус имеют приоритет на  движение.
n   На остановках запрещены: движение задним ходом, разворот и стоянки.
 
ПРИОРИТЕТ МАРШРУТНЫХ Т.С.
 
n  Трамвай имеет приоритет на движение при прочих равных условиях.
